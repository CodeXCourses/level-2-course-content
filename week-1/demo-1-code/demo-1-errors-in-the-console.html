Errors in the console.

When you click the buttons, you will see various error codes. Take a look at the line numbers in the console and examine what the error code means.

Example

      Cannot read properties of null

This error occurs if a variable is null and you try to access a property, as though it were an object.

Example

      Cannot read properties of null

This error can occur if a function returns null. For example if you misspell the id when you call document.querySelector or document.getElementId

Example

      Cannot read properties of undefined

This error can occur if you forget to define a variable. Since variables start as undefined, if you don't set it with an = sign, it will be undefined. Trying to treat undefined as an object causes an error.
