Errors in HTML

Some errors in HTML can be viewed in the console.

Open you console and have a look at some of the errors.

Refused to apply style (CSS)

The first error should be something like:

    Refused to apply style from 'http://...' because its MIME type ('text/html') is not a supported stylesheet MIME type, and strict MIME checking is enabled.

Look carefully. Should it be sytlin.css or style.css?

Failed to load resource (img)

If you misspell the image name, you will get an error like this:

    Failed to load resource: the server responded with a status of 404 (Not Found) stylin.css:1

This image is blank! Check the console. What is the error?

Check the directory, it should be image.jpg, not imagine.jpg. Oops!
